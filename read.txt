1. create flutter module
flutter create -t module --org com.example flutter_module

2.根目录下的local.properties中增加
flutter.sdk=/Users/malin/my_github/flutter

3.拉取flutter的依赖
cd flutter_module/.android
flutter pub get

4.编译flutter module
cd flutter_module
flutter run lib/main.dart

[flutter-cli](https://docs.flutter.dev/reference/flutter-cli)

3.flutter_module/.android/local.properties 中增加

sdk.dir=/Users/malin/Library/Android/sdk
flutter.sdk=/Users/malin/my_github/flutter
flutter.buildMode=debug
flutter.versionName=1.0.0
flutter.versionCode=1

3.open flutter android project
open flutter_module/.android/build.gradle
